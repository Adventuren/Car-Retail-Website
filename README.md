Welcome to the Car Retail Website repository! This project is a web application built with .NET Core MVC and WebAPI, featuring a user-friendly interface for customers to explore and purchase cars, as well as an administration panel for managing inventory and orders.

Technologies Used
.NET Core MVC: For building the web application.
WebAPI: To support communication between the frontend and backend.
Entity Framework Core: For database interactions and management.
HTML, CSS, JavaScript: For the frontend development.
